Overview
AgriSmart AI is an intelligent agriculture support system that integrates machine learning with real-time environmental data to assist farmers in making better decisions. The system focuses on two major aspects:
💧 Smart Irrigation Recommendation
📈 Crop Price Prediction
It helps improve water efficiency, increase yield, and predict market trends for better profitability.

🚀 Features
🌾 Smart Irrigation
Predicts irrigation requirements based on:
Temperature
Humidity
Soil pH
Rainfall
Crop Type
Provides suggestions like:
“High Water Needed”
“Irrigation Not Required”
📊 Crop Price Prediction
Predicts future crop prices using:
State, District, Market details
Commodity & Variety codes
Date (Day, Month, Year)
Min & Max prices
Displays:
Current Price
Next Month Forecast
Price Trend Graph

📉 Data Visualization
Interactive graph showing price trends over time
Helps farmers understand market behavior
